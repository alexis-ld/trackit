## Installation

#### #1 Clone TrackIt
```sh
$> git clone git@github.com:trackit/trackit.git
```

#### #2 Start TrackIt
```sh
$> docker-compose up
```

#### #3 Let's go
Once TrackIt is started, go to [http://localhost/](http://localhost/) and let TrackIt optimize your cloud!
